# Quality Scoring Quick Reference Card

**100-Point Content Quality Assessment**

---

## Scoring System Overview

| Score | Grade | Status |
|-------|-------|--------|
| 90-100 | A+ | Excellent - Publish immediately |
| 80-89 | A | Good - Minor tweaks needed |
| 70-79 | B | Fair - Moderate improvements |
| 60-69 | C | Poor - Major revisions needed |
| < 60 | F | Fail - Rewrite required |

**Minimum publish threshold: 80 points**

---

## 1. SEO Fundamentals (25 points)

### Title Tag (5 points)
- [ ] 50-60 characters (2 pts)
- [ ] Includes primary keyword (2 pts)
- [ ] Compelling and click-worthy (1 pt)

### Meta Description (5 points)
- [ ] 150-160 characters (2 pts)
- [ ] Includes primary keyword (1 pt)
- [ ] Includes CTA (1 pt)
- [ ] Unique across site (1 pt)

### URL Structure (3 points)
- [ ] Short and descriptive (1 pt)
- [ ] Includes keyword (1 pt)
- [ ] No special characters (1 pt)

### Header Tags (5 points)
- [ ] One H1 with keyword (2 pts)
- [ ] H2s structure content logically (2 pts)
- [ ] H3s for subsections (1 pt)

### Keyword Usage (7 points)
- [ ] Keyword in first 100 words (2 pts)
- [ ] Keyword density 0.5-2% (2 pts)
- [ ] LSI keywords used naturally (2 pts)
- [ ] No keyword stuffing (1 pt)

---

## 2. Content Quality (25 points)

### Depth & Value (10 points)
- [ ] Comprehensive coverage (3 pts)
- [ ] Actionable information (3 pts)
- [ ] Unique insights (2 pts)
- [ ] Updated/current info (2 pts)

### Writing Quality (8 points)
- [ ] Clear and concise (2 pts)
- [ ] No grammar/spelling errors (2 pts)
- [ ] Active voice (1 pt)
- [ ] Scannable (short paragraphs) (1 pt)
- [ ] Appropriate reading level (2 pts)

### Length (4 points)
- [ ] 1,000+ words (1 pt)
- [ ] 1,500+ words (2 pts)
- [ ] 2,000+ words (3 pts)
- [ ] 2,500+ words (4 pts)

### Engagement (3 points)
- [ ] Hook in intro (1 pt)
- [ ] Examples/stories (1 pt)
- [ ] Strong conclusion/CTA (1 pt)

---

## 3. User Experience (20 points)

### Formatting (6 points)
- [ ] Bullet points/lists (2 pts)
- [ ] Bold/italic for emphasis (1 pt)
- [ ] White space (not cramped) (1 pt)
- [ ] Logical content flow (2 pts)

### Media (8 points)
- [ ] At least 1 image (2 pts)
- [ ] Images optimized (< 100KB) (2 pts)
- [ ] Alt text on all images (2 pts)
- [ ] Video/infographic (bonus 2 pts)

### Navigation (6 points)
- [ ] Table of contents (2 pts)
- [ ] Breadcrumbs (1 pt)
- [ ] Jump links to sections (1 pt)
- [ ] Related content links (2 pts)

---

## 4. E-E-A-T Signals (15 points)

### Experience (4 points)
- [ ] Real-world examples (2 pts)
- [ ] Specific details/numbers (1 pt)
- [ ] Personal insights (1 pt)

### Expertise (4 points)
- [ ] Author bio displayed (2 pts)
- [ ] Author credentials mentioned (2 pts)

### Authoritativeness (4 points)
- [ ] External links to authorities (2 pts)
- [ ] Citations for claims (2 pts)

### Trustworthiness (3 points)
- [ ] Sources cited (1 pt)
- [ ] Data is current (1 pt)
- [ ] Contact info visible (1 pt)

---

## 5. Technical SEO (10 points)

### Page Speed (3 points)
- [ ] Load time < 3 seconds (3 pts)
- [ ] Load time < 5 seconds (2 pts)
- [ ] Load time < 7 seconds (1 pt)

### Mobile Optimization (3 points)
- [ ] Responsive design (2 pts)
- [ ] Touch-friendly buttons (1 pt)

### Structured Data (2 points)
- [ ] Schema markup implemented (1 pt)
- [ ] Schema validated (1 pt)

### Security & Accessibility (2 points)
- [ ] HTTPS (1 pt)
- [ ] Accessible (WCAG AA) (1 pt)

---

## 6. Internal Linking (5 points)

- [ ] 3+ relevant internal links (2 pts)
- [ ] Contextual anchor text (1 pt)
- [ ] Links to high-authority pages (1 pt)
- [ ] Links from high-authority pages (1 pt)

## Priority Fixes by Score Range

### 70-79 Points (Most Common Issues)
**Fix these first:**
1. Add more internal links (quick win)
2. Expand content depth (add 300-500 words)
3. Add E-E-A-T signals (author bio, citations)
4. Optimize images (compress, add alt text)
5. Add table of contents

**Time needed**: 1-2 hours

### 60-69 Points (Major Issues)
**Critical fixes:**
1. Rewrite title/meta for SEO
2. Restructure with proper headers
3. Add 500-1000 words of value
4. Add media (images/video)
5. Implement structured data
6. Fix keyword usage

**Time needed**: 3-4 hours

### < 60 Points (Rewrite Needed)
**Start over or major overhaul:**
1. Research competitors (what ranks?)
2. Create detailed outline
3. Write comprehensive content
4. Follow all checklist items
5. Get editor review

**Time needed**: 6-8 hours

## Automated vs Manual Scoring

### Automated Tools Check:
- Technical SEO (page speed, mobile, HTTPS)
- SEO fundamentals (title, meta, headers)
- Readability scores
- Image optimization

### Manual Review Required:
- Content depth and value
- E-E-A-T signals
- Writing quality
- User engagement elements
- Strategic internal linking

**Best practice**: Use automated tools first, then manual review.

---

## Common Deductions

### -5 Points
- No author bio
- No table of contents (long content)
- Slow page load (> 5 seconds)
- No structured data

### -3 Points
- Weak meta description
- No images
- Poor header structure
- Thin content (< 1,000 words)

### -2 Points
- Missing internal links
- No external authority links
- Non-optimized images
- No alt text

### -1 Point
- Minor formatting issues
- Single grammar error
- Missing breadcrumbs
- Weak CTA

## Quality Improvement ROI

### High ROI Improvements (Do First)
1. Add internal links (+5 pts, 10 min)
2. Optimize title tags (+3 pts, 15 min)
3. Add table of contents (+2 pts, 10 min)
4. Compress images (+2 pts, 15 min)
5. Add author bio (+2 pts, 5 min)

### Medium ROI Improvements
1. Expand content (+4 pts, 1-2 hours)
2. Add citations (+2 pts, 30 min)
3. Implement schema (+2 pts, 20 min)
4. Add related content (+2 pts, 30 min)

### Low ROI Improvements
1. Perfect every minor detail (+1-2 pts, 2+ hours)
2. Excessive content length (+1 pt, 2+ hours)
3. Advanced media creation (+2 pts, 4+ hours)

**Focus on high ROI improvements first!**

---

**Remember**: Quality is not perfection. Ship 80+ content consistently rather than pursuing 100-point unicorns.
