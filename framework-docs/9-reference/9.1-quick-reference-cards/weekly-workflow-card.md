# Weekly Workflow Quick Reference Card

**Print this page for weekly planning sessions**

---

## Monday: Planning & Strategy (2-3 hours)

### 1. Analytics Deep Dive (45 min)
```bash
/generate-monthly-report
# Focus on:
# - Traffic trends (week over week)
# - Top performing content
# - Keyword ranking changes
# - Conversion rate changes
```

**Key Questions:**
- Which content drove most traffic last week?
- Which keywords moved up/down?
- What's our best converting page?
- Any technical issues detected?

### 2. Content Calendar Planning (45 min)
- [ ] Review last week's publications
- [ ] Plan this week's topics (5-7 posts)
- [ ] Assign keywords to each post
- [ ] Schedule publication dates
- [ ] Identify hub page opportunities

### 3. Competitor Analysis (30 min)
```bash
/competitor-analysis [competitor-domain]
# Check top 3 competitors
# Note their new content
# Identify content gaps
```

### 4. Set Weekly Goals (15 min)
- Content to publish: [5-7 posts]
- Content to refresh: [3-5 posts]
- Links to acquire: [2-5 links]
- Hub pages to build: [0-1]

---

## Tuesday-Thursday: Execution (4-6 hours/day)

### Content Production
- **Morning**: Write 1-2 new posts
- **Afternoon**: Refresh 1 existing post
- **End of day**: Outreach for links

### Daily Checklist
- [ ] Publish 1-2 new posts
- [ ] Refresh 1 existing post
- [ ] Add 5-10 internal links
- [ ] Send 5-10 outreach emails
- [ ] Review and respond to comments

---

## Friday: Optimization & Quality (3-4 hours)

### 1. Content Audit (1 hour)
```bash
/audit-content-performance
# Identify:
# - Underperforming content (refresh candidates)
# - High-potential content (amplify)
# - Outdated content (update or remove)
```

### 2. Technical SEO Check (1 hour)
- [ ] Fix broken links (internal/external)
- [ ] Update outdated statistics
- [ ] Optimize slow-loading pages
- [ ] Check mobile usability
- [ ] Verify structured data

### 3. Link Building Review (1 hour)
- [ ] Track outreach responses
- [ ] Follow up on pending outreach
- [ ] Document acquired links
- [ ] Identify new prospect sources

### 4. Plan Next Week (30 min)
- Review what worked/didn't work
- Adjust strategy if needed
- Prepare research for next week
- Set specific targets
