# Client Onboarding Checklist

**Complete this checklist for every new SEO client**

---

## Phase 1: Discovery (Week 1)

### Client Information
- [ ] Company name and legal entity
- [ ] Industry and niche
- [ ] Primary location(s)
- [ ] Service area (local/regional/national/global)
- [ ] Years in business
- [ ] Team size
- [ ] Annual revenue range
- [ ] Key decision makers (names, roles, emails)

### Business Goals
- [ ] Primary business objectives
- [ ] Revenue targets (6-12 months)
- [ ] Growth rate expectations
- [ ] Market expansion plans
- [ ] Competitive positioning goals
- [ ] Brand awareness goals

### SEO History
- [ ] Previous SEO efforts (what worked/didn't work)
- [ ] Current organic traffic levels
- [ ] Current keyword rankings
- [ ] Previous penalties or issues
- [ ] Prior agency relationships
- [ ] Existing content inventory
- [ ] Current backlink profile

### Target Audience
- [ ] Primary customer demographics
- [ ] Secondary customer segments
- [ ] Customer pain points
- [ ] Buying journey stages
- [ ] Geographic targeting
- [ ] Language preferences

---

## Phase 2: Technical Setup (Week 1-2)

### Access & Credentials
- [ ] Website admin access (WordPress, etc.)
- [ ] Google Analytics access (Editor role)
- [ ] Google Search Console access (Owner role)
- [ ] Google My Business access (if local)
- [ ] CMS login credentials
- [ ] Hosting account access
- [ ] Domain registrar access
- [ ] Email account for communications
- [ ] Social media accounts (optional)

### Tool Setup
- [ ] Add domain to Ahrefs/SEMrush
- [ ] Set up rank tracking (target keywords)
- [ ] Configure uptime monitoring
- [ ] Set up page speed monitoring
- [ ] Install heat mapping (if needed)
- [ ] Set up conversion tracking
- [ ] Configure Search Console alerts
- [ ] Set up Analytics reports

### Technical Audit
- [ ] Crawl site with Screaming Frog
- [ ] Check site speed (GTmetrix)
- [ ] Mobile usability test (Google)
- [ ] Security check (HTTPS, SSL)
- [ ] Check robots.txt
- [ ] Verify XML sitemap
- [ ] Check canonical tags
- [ ] Identify duplicate content
- [ ] Check for 404 errors
- [ ] Review site structure
- [ ] Check internal linking
- [ ] Verify structured data

---

## Phase 3: Research (Week 2)

### Competitor Analysis
- [ ] Identify top 5-10 competitors
- [ ] Analyze competitor keywords
- [ ] Review competitor content
- [ ] Analyze competitor backlinks
- [ ] Check competitor site structure
- [ ] Note competitor strengths
- [ ] Identify competitor weaknesses
- [ ] Find content gaps

### Keyword Research
- [ ] Brainstorm seed keywords (50-100)
- [ ] Expand with keyword tools (500-1,000)
- [ ] Categorize by intent (INCT framework)
- [ ] Assign difficulty scores
- [ ] Estimate search volumes
- [ ] Map to existing pages
- [ ] Identify quick wins (low competition)
- [ ] Prioritize target keywords (top 50)
- [ ] Create keyword tracking list

### Content Audit
- [ ] Inventory all existing pages
- [ ] Assign keywords to pages
- [ ] Score content quality (1-100)
- [ ] Identify thin content (< 300 words)
- [ ] Find duplicate content
- [ ] Mark outdated content
- [ ] Note underperforming pages
- [ ] Identify refresh opportunities
- [ ] Flag pages for deletion
- [ ] List content gaps

---

## Phase 4: Strategy Development (Week 3)

### SEO Strategy Document
- [ ] Executive summary
- [ ] SWOT analysis
- [ ] Target keyword list (top 50)
- [ ] Content strategy overview
- [ ] Link building strategy
- [ ] Technical SEO priorities
- [ ] Timeline and milestones
- [ ] Success metrics
- [ ] Reporting schedule

### Content Strategy
- [ ] Define content pillars (3-5)
- [ ] Identify hub page topics (5-10)
- [ ] Plan spoke pages per hub (10-20 each)
- [ ] Create content calendar (3 months)
- [ ] Define content types (blog, guides, etc.)
- [ ] Set publishing frequency
- [ ] Assign content creation responsibilities
- [ ] Define content quality standards

### Link Building Strategy
- [ ] Define target DA/DR thresholds
- [ ] Identify link building tactics
- [ ] Create prospect research plan
- [ ] Define outreach templates
- [ ] Set monthly link targets
- [ ] Identify existing link opportunities
- [ ] Plan relationship building

### Technical SEO Roadmap
- [ ] Prioritize technical fixes
- [ ] Create implementation timeline
- [ ] Assign responsibilities
- [ ] Define success criteria
- [ ] Set up monitoring

---

## Phase 5: Kickoff (Week 3-4)

### Kickoff Meeting Preparation
- [ ] Prepare presentation deck
- [ ] Create strategy document
- [ ] Prepare competitive analysis
- [ ] Prepare keyword research report
- [ ] Create content calendar preview
- [ ] Prepare technical audit summary
- [ ] Set meeting agenda
- [ ] Send calendar invite

### Kickoff Meeting Agenda
- [ ] Introductions (15 min)
- [ ] Business goals review (15 min)
- [ ] Strategy presentation (30 min)
- [ ] Content calendar review (15 min)
- [ ] Timeline and milestones (15 min)
- [ ] Roles and responsibilities (15 min)
- [ ] Communication plan (15 min)
- [ ] Q&A (15 min)

### Post-Kickoff Actions
- [ ] Send meeting notes
- [ ] Send strategy document
- [ ] Share access to project management tool
- [ ] Set up regular meeting schedule
- [ ] Send first content assignments
- [ ] Begin technical implementations
- [ ] Start content creation

---

### Quick Wins (Week 1-2)

- [ ] Optimize existing high-traffic pages
- [ ] Add missing meta descriptions
- [ ] Optimize title tags
- [ ] Add internal links
- [ ] Fix broken links
- [ ] Optimize images (compress, alt text)
- [ ] Implement schema markup

### Content Creation (Ongoing)
- [ ] Publish 4-8 new posts (Month 1)
- [ ] Refresh 2-4 existing posts
- [ ] Create 1 hub page
- [ ] Implement content calendar
- [ ] Set up editorial workflow
- [ ] Define approval process

### Link Building (Ongoing)
- [ ] Research 50+ prospects
- [ ] Launch outreach campaign
- [ ] Track responses
- [ ] Document acquired links
- [ ] Build relationship pipeline

## Red Flags to Watch For

### During Discovery
- [ ] Unrealistic expectations ("rank #1 in 1 month")
- [ ] Previous black hat SEO tactics
- [ ] History of Google penalties
- [ ] Unwillingness to invest in content
- [ ] Poor site technical foundation
- [ ] No clear business goals
- [ ] Excessive decision-makers
- [ ] Micromanagement tendencies

### During Kickoff
- [ ] Scope creep requests
- [ ] Resistance to strategy
- [ ] Lack of trust in expertise
- [ ] Unreasonable timeline demands
- [ ] Budget constraints vs. expectations
- [ ] Poor internal communication

**If 3+ red flags: Consider declining project or adjusting scope/pricing**
