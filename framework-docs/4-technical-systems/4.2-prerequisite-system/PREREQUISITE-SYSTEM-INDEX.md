### For Command Users
You're running commands and encountered prerequisite errors.

**Primary Resources:**
1. **Error Messages** - (shown in terminal)
   - What's missing

   - Step-by-step instructions

3. **Quick Reference** - `PREREQUISITE-QUICK-REFERENCE.md` → Section 11
   - Example workflows
   - Troubleshooting

**Workflow:**
```
1. Run validation before command
2. If errors, read error messages
3. Follow recovery plan
4. Re-run validation
5. Execute command
```
