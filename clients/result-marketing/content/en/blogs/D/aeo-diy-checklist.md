## **Summary: 90-Day AEO Checklist**

**Key Takeaways:**
- Week 1-4: Technical foundation (schema, GSC, baseline metrics)
- Week 5-8: Content creation (FAQ pages, question-targeted articles)
- Week 9-12: Optimization (citation monitoring, competitor analysis, adjustments)
- By Day 90: 400-600 AI citations, first conversions from AEO

**Read Time:** 7 minutes | **Category:** DIY vs Hire

---

## **Direct Answer: AEO DIY Checklist Overview**

This 90-day AEO checklist guides you through three phases: Foundation (Weeks 1-4), Content Creation (Weeks 5-8), and Optimization (Weeks 9-12). Complete 4-6 tasks weekly (3-5 hours total) to build 400-600 AI citations in 90 days. Essential tasks include FAQ schema implementation, 12 question-targeted articles, 50 query monitoring, and competitor citation tracking. Track completion in a spreadsheet or project management tool. Expect 60-70% of agency results with consistent execution.

---

## [Phase 1: Foundation (Weeks 1-4)](#phase-1-foundation-weeks-1-4)

### Week 1: Setup + Baseline

**Task 1.1: Install Schema Plugin** [30 minutes]
- [ ] Install RankMath, Yoast SEO, or Schema Pro
- [ ] Activate plugin
- [ ] Complete initial setup wizard
- [ ] Verify installation

**Task 1.2: Organization Schema** [1 hour]
- [ ] Gather business info (name, address, phone, logo)
- [ ] Add organization schema to homepage
- [ ] Test with Google Rich Results Test
- [ ] Fix any errors

**Task 1.3: Google Search Console** [45 minutes]
- [ ] Verify ownership of site
- [ ] Submit XML sitemap
- [ ] Check for existing manual penalties
- [ ] Note current organic traffic baseline

**Task 1.4: Baseline Metrics** [45 minutes]
- [ ] Record current organic traffic (GA4)
- [ ] Record current branded search volume (GSC)
- [ ] Manually check 20 core queries for AI citations
- [ ] Create baseline tracking spreadsheet

**Week 1 Total Time: 3 hours**

---

### Week 2: FAQ Schema Implementation

**Task 2.1: Identify Top 50 Customer Questions** [1 hour]
- [ ] Review sales emails/calls for common questions
- [ ] Check competitor FAQs
- [ ] Use AnswerThePublic (free tier) for question ideas
- [ ] Organize by category (pricing, process, quality, location)

**Task 2.2: FAQ Schema - Page 1** [1.5 hours]
- [ ] Select top service page
- [ ] Write 10-15 FAQs with answers (50-75 words each)
- [ ] Add FAQ schema using plugin
- [ ] Test with Rich Results Test
- [ ] Submit to GSC for indexing

**Task 2.3: FAQ Schema - Page 2** [1.5 hours]
- [ ] Select second service page
- [ ] Write 10-15 FAQs
- [ ] Add FAQ schema
- [ ] Test + submit to GSC

**Task 2.4: FAQ Schema - Page 3** [1.5 hours]
- [ ] Select third service page
- [ ] Write 10-15 FAQs
- [ ] Add FAQ schema
- [ ] Test + submit to GSC

**Week 2 Total Time: 5.5 hours**

---

### Week 3: Content Planning

**Task 3.1: Keyword Research** [1.5 hours]
- [ ] Use Google Trends for trending questions
- [ ] Check GSC for existing impression queries
- [ ] Identify 50 core questions to target
- [ ] Prioritize by search volume + AI Overview presence

**Task 3.2: Content Calendar Creation** [1 hour]
- [ ] Plan 12 articles (3 per month for 4 months)
- [ ] Assign topics: 4 "how much", 4 "how to", 4 "vs"
- [ ] Set deadlines in calendar
- [ ] Schedule reminders

**Task 3.3: Competitor Citation Audit** [1.5 hours]
- [ ] Search 30 core queries
- [ ] Note who's cited in AI Overviews
- [ ] Identify citation patterns (tables, lists, Q&A)
- [ ] Document in spreadsheet

**Task 3.4: Google Business Profile Optimization** [1 hour]
- [ ] Add business description with service keywords
- [ ] Update services list
- [ ] Add 10 FAQs to GBP Q&A section
- [ ] Upload 5 new photos

**Week 3 Total Time: 5 hours**

---

### Week 4: First Content Creation

**Task 4.1: Article 1 - "How Much"** [2 hours]
- [ ] Choose "how much does [service] cost" topic
- [ ] Research current pricing
- [ ] Write 800-1,000 word article
- [ ] Include price range table
- [ ] Add FAQ schema (3-5 questions)
- [ ] Publish + submit to GSC

**Task 4.2: Article 2 - "How To"** [2 hours]
- [ ] Choose "how to [achieve outcome]" topic
- [ ] Outline step-by-step process
- [ ] Write 1,000-1,200 word guide
- [ ] Add Article schema
- [ ] Publish + submit to GSC

**Task 4.3: Article 3 - Comparison** [2 hours]
- [ ] Choose "[service] vs [alternative]" topic
- [ ] Create comparison table
- [ ] Write 800-1,000 word comparison
- [ ] Add FAQ schema
- [ ] Publish + submit to GSC

**Task 4.4: Weekly Citation Check** [1 hour]
- [ ] Search 50 core queries
- [ ] Note any new AI citations for your content
- [ ] Track competitor changes
- [ ] Update spreadsheet

**Week 4 Total Time: 7 hours**

---

## [Phase 2: Content Creation (Weeks 5-8)](#phase-2-content-creation-weeks-5-8)

### Week 5-8 Weekly Template

**Each Week Complete:**

**Content Creation (2.5 hours)**
- [ ] Write 1 article (rotate: how much, how to, vs)
- [ ] Add appropriate schema
- [ ] Optimize for AI citation format (tables, lists)
- [ ] Publish + submit to GSC

**Citation Monitoring (1 hour)**
- [ ] Search 50 core queries
- [ ] Track your citation count
- [ ] Note competitor patterns
- [ ] Adjust content strategy based on findings

**Content Update (30 min)**
- [ ] Update 1 existing article
- [ ] Add fresh data/examples
- [ ] Improve based on citation patterns
- [ ] Re-submit to GSC

**Weekly Total: 4 hours**

**By End of Week 8:**
- [ ] 12 total articles published
- [ ] 150-250 AI citations
- [ ] Branded search up 20-40%

---

## [Phase 3: Optimization (Weeks 9-12)](#phase-3-optimization-weeks-9-12)

### Week 9: Performance Review

**Task 9.1: GA4 Deep Dive** [1 hour]
- [ ] Review organic traffic changes
- [ ] Analyze branded search growth
- [ ] Identify top-performing articles
- [ ] Note traffic sources

**Task 9.2: GSC Analysis** [1 hour]
- [ ] Check impression growth
- [ ] Identify new queries triggering impressions
- [ ] Review average position changes
- [ ] Note click-through rate

**Task 9.3: Citation Count** [1 hour]
- [ ] Count total AI citations (all queries)
- [ ] Compare to Week 4 baseline
- [ ] Calculate growth rate
- [ ] Identify best-performing content types

**Task 9.4: ROI Calculation** [1 hour]
- [ ] Track leads from AEO content
- [ ] Calculate attributed revenue
- [ ] Compare AEO investment (time + tools)
- [ ] Assess ROI

**Week 9 Total Time: 4 hours**

---

### Week 10-12: Scale + Refine

**Weekly Tasks:**

**Content Gap Analysis** [1 hour]
- [ ] Identify questions not yet answered
- [ ] Check competitor citations for gaps
- [ ] Prioritize new topics
- [ ] Update content calendar

**Advanced Schema** [1 hour]
- [ ] Add Article schema to all guides
- [ ] Add Review schema (if testimonials exist)
- [ ] Add LocalBusiness schema (if not done)
- [ ] Test all schemas

**Link Building** [1 hour]
- [ ] Identify 10 citation opportunities
- [ ] Submit to 5 local directories
- [ ] Reach out to 3 industry sites for guest posts
- [ ] Track new backlinks

**Conversion Optimization** [30 min]
- [ ] Add CTAs to top articles
- [ ] Optimize contact form placement
- [ ] Test WhatsApp button visibility
- [ ] Review mobile experience

**Weekly Total: 3.5 hours**

## [Common Roadblocks + Solutions](#common-roadblocks-solutions)

### Roadblock 1: "I Don't Have Time"

**Solution:**
- Batch content creation (write 3 articles in one session)
- Use AI tools (ChatGPT) for first drafts
- Set recurring calendar blocks (same time each week)
- Hire freelancer for writing (RM150-300/article)

---

### Roadblock 2: "Not Technical Enough for Schema"

**Solution:**
- Use plugins (RankMath, Yoast) - no coding needed
- Copy-paste templates from Schema.org
- Watch YouTube tutorials (30 min to learn basics)
- Hire freelancer for one-time setup (RM500-1,000)

---

### Roadblock 3: "Not Seeing Results Yet"

**Solution:**
- AEO takes 60-90 days for first citations
- Focus on citation count, not traffic initially
- Track branded search growth (leading indicator)
- Compare to baseline, not competitors yet

---

### Roadblock 4: "Running Out of Content Ideas"

**Solution:**
- AnswerThePublic for question ideas
- Check "People Also Ask" in Google
- Ask sales team for customer questions
- Repurpose existing content into new formats

## **FAQ: AEO DIY Checklist**

### **Can I shorten this 90-day timeline?**

Yes, if you invest more time. 10-15 hours/week can achieve in 60 days what takes 90 days at 3-5 hours/week. But most businesses burn out at that pace. 90 days at sustainable intensity is better than 60 days then quitting.

### **What if I miss a week?**

Don't stress. Pick up where you left off. Missing one week adds 7 days to your timeline, not failure. The key is consistency over 90 days, not perfection every week. 11 out of 12 weeks completed still gets results.

### **Do I need to complete every single task?**

Core tasks are mandatory (schema, content creation, citation monitoring). Nice-to-haves (link building, advanced schema) can wait. Focus on the 80/20: 20% of tasks (FAQ schema, question articles, monitoring) deliver 80% of results.

### **Can I hire someone to execute this checklist for me?**

Yes. Freelancers on Upwork charge RM500-1,500/month to execute. Or hire an agency (RM1,200-3,000/month) for professional execution with better tools and experience. This checklist is what agencies execute, just DIY-ified.

### **What happens after Day 90?**

Assess results. If you have 400-600 citations and positive ROI, continue with maintenance mode (2-3 hours/month). If results are weak, audit strategy (maybe targeting wrong queries) or hire professional help. Day 90 is decision point, not finish line.
