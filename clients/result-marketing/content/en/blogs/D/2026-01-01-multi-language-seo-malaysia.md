## How Multi-Language SEO Works

### Technical Implementation

**We use hreflang tags.**

**What are hreflang tags?**
Code that tells Google: "This page has versions in other languages."

**Example:**
```html
<link rel="alternate" hreflang="en" href="domain.com/en/crane-rental" />
<link rel="alternate" hreflang="zh" href="domain.com/zh/Ëµ∑ÈáçÊú∫ÁßüËµÅ" />
<link rel="alternate" hreflang="ms" href="domain.com/ms/penyewaan-kraan" />
```

**Google sees:**
- English version exists
- Chinese version exists
- Malay version exists
- Shows correct version based on searcher's language

**Without hreflang:**
Google gets confused, might:
- Show wrong language version
- Flag as duplicate content
- Not rank any version well

### URL Structure
