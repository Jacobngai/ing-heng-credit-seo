## **Implementation Strategy**

### Separate URL Structure

**Recommended approach:**

```
example.com/en/ (English)
example.com/zh/ (Chinese)
example.com/ms/ (Malay)
```

**Alternative (subdomains):**
```
en.example.com
zh.example.com
ms.example.com
```

**Language detection:**
- Auto-detect browser language
- Language switcher on site
- Default to English if unknown

## **Hreflang and Language Tags**

### Hreflang Implementation

**Tell search engines which language each page is in:**

```html
<link rel="alternate" hreflang="en" href="https://example.com/en/seo-services" />
<link rel="alternate" hreflang="zh" href="https://example.com/zh/seo-services" />
<link rel="alternate" hreflang="ms" href="https://example.com/ms/seo-services" />
<link rel="alternate" hreflang="x-default" href="https://example.com/en/seo-services" />
```

**Why:** Prevents duplicate content issues. Helps AI engines serve correct language.
