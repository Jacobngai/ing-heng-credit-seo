## FAQ Schema Deep Dive

FAQ schema is the highest-impact schema for AI search.

**Why It Works:**
- Direct question-answer format
- AI loves Q&A content
- Easy to extract and cite
- Multiple citations per page

**Best Practices:**
- Use real customer questions
- Provide direct answers
- Keep answers concise (50-100 words)
- Update regularly
- Include specific data

**Example:**
```json
{
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "How much does crane rental cost?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "RM800-3,500 per day depending on capacity..."
    }
  }
]
}
```

**Real Results:**
- Equipment rental client
- Added FAQ schema to 10 pages
- AI citations: +35/month
- Traffic: +85%
- Leads: +65%
