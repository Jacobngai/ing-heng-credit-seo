## **What is Visual Search?**

### Definition

**Visual search** = Using images instead of text to search.

**How it works:**
1. User takes photo or uploads image
2. AI analyzes image (objects, text, context)
3. AI searches for similar images/products
4. AI provides results with sources

**Popular tools:**
- **Google Lens** (integrated with Google Photos, Android)
- **Pinterest Lens** (shopping inspiration)
- **Bing Visual Search** (product search)

## **Image Optimization for AI**

### 1. High-Quality Images

**Requirements:**
- Resolution: 1200x630px minimum (OG standard)
- Format: WebP preferred (smaller file size)
- Quality: Professional, clear lighting
- Subject: Central, focused, uncluttered

**Product images:**
- White background
- Multiple angles (3-5 photos)
- Lifestyle photos (product in use)
- Size reference (show scale)

### 2. Descriptive Filenames

**Bad:**
`IMG_1234.jpg`
`photo.png`

**Good:**
`seo-services-malaysia-pricing.jpg`
`kuala-lumpur-plumber-emergency-service.jpg`

**Why:** AI engines use filenames to understand image content.

### 3. Alt Text

**Describe images accurately:**

```html
<img src="plumber-kuala-lumpur.jpg" alt="Emergency plumber in Kuala Lumpur fixing burst pipe in residential bathroom" />
```

**Best practices:**
- Descriptive (what's in the image)
- Include keywords naturally
- Under 125 characters
- Accurate (don't keyword stuff)

## **Google Lens Optimization**

### Make Your Content Lens-Discoverable

**Strategies:**

**1. Add text to images**
- Product names on product photos
- Prices, specifications visible
- Location names on building photos
- Watermark with brand name

**2. Use consistent branding**
- Logo on all images
- Consistent style/filter
- Brand colors prominent

**3. Optimize for products**
- Clean, white backgrounds
- Multiple angles
- Size reference
- Contextual images (product in use)

**Why:** Google Lens can read text in images and recognize branded products.

## **Image Schema Markup**

### ImageObject Schema

```json
{
  "@type": "ImageObject",
  "name": "Emergency Plumber Kuala Lumpur",
  "caption": "Professional plumber repairing burst pipe in Kuala Lumpur residential bathroom",
  "description": "ABC Plumbing provides 24/7 emergency plumbing services in Kuala Lumpur",
  "author": {
    "@type": "Organization",
    "name": "ABC Plumbing"
  },
  "contentUrl": "https://site.com/images/plumber-kl.jpg",
  "uploadDate": "2026-01-09"
}
```

**Why:** Helps AI engines understand image context and details.
