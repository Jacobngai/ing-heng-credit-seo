## **Essential Free Tools (Must-Have)**

### Tool 1: Google Search Console

**Cost:** Free
**URL:** https://search.google.com/search-console

**What it does:**
- Shows which keywords your site ranks for
- Tracks impressions, clicks, CTR, average position
- Identifies indexing issues
- Submits URLs for crawling
- Mobile usability reports

**How to use for AEO:**

**Step 1: Check current performance**
- Go to "Performance" report
- Filter by "Queries"
- Sort by impressions
- Note: Which keywords trigger impressions?
- Note: Which pages get impressions?
- This is your baseline

**Step 2: Find AI Overview opportunities**
- Look for question-based queries
- Example: "how much crane rental cost"
- If you're ranking positions 5-20 for these
- You're close to AI Overview citations
- Optimize these pages first

**Step 3: Track citation impact**
- When AI Overview cites your content
- Check Search Console 7 days later
- Look for impression/click increases
- Correlate citations with traffic

**Why it's essential:**
Google's own data. Most accurate source of how Google sees your site.

### Tool 2: Google Analytics 4

**Cost:** Free
**URL:** https://analytics.google.com

**What it does:**
- Tracks website traffic
- Measures user behavior
- Conversion tracking
- Traffic source attribution

**How to use for AEO:**

**Set up conversion tracking:**
- [ ] Contact form submissions
- [ ] WhatsApp button clicks
- [ ] Phone number clicks (mobile)
- [ ] Email link clicks

**Create custom segment:**
- [ ] "Organic traffic" segment
- [ ] Filter: Medium = "organic"
- [ ] Compare: Before vs after AEO efforts

**Track attribution:**
- [ ] Which blog posts drive conversions?
- [ ] Which AI Overview citations generate leads?
- [ ] Calculate ROI: Organic leads Ã— value

**Why it's essential:**
Proves AEO ROI. If you can't measure conversions, you can't measure success.

### Tool 3: Google Rich Results Test

**Cost:** Free
**URL:** https://search.google.com/test/rich-results

**What it does:**
- Tests schema markup
- Shows which rich results your page eligible for
- Identifies schema errors
- Live preview of how your page appears in search

**How to use for AEO:**

**After adding schema to any page:**
1. Paste URL into Rich Results Test
2. Check for errors
3. Fix any issues
4. Re-test until all clear
5. Then submit to Search Console

**What to test:**
- FAQ schema (must-have for AEO)
- Organization schema
- Article schema
- Review schema
- LocalBusiness schema

**Why it's essential:**
Broken schema = no AI citations. This tool prevents that.

### Tool 4: Google Schema Markup Helper

**Cost:** Free
**URL:** https://www.google.com/searchconsole/markup-helper

**What it does:**
- Visual schema markup generator
- Highlights data on your page
- Generates JSON-LD code
- No coding required

**How to use for AEO:**

**For FAQ schema:**
1. Enter your page URL
2. Choose "FAQ" data type
3. Use mouse to highlight questions on page
4. Highlight answers
5. Tool generates JSON-LD
6. Copy-paste into your website `<head>` section

**For Article schema:**
1. Enter blog post URL
2. Highlight headline, author, date, publisher
3. Tool generates structured data
4. Add to page

**Why it's essential:**
Makes schema accessible to non-coders. DIY-friendly.

### Tool 5: Google PageSpeed Insights

**Cost:** Free
**URL:** https://pagespeed.web.dev

**What it does:**
- Tests page speed (mobile and desktop)
- Identifies performance issues
- Suggests fixes
- Scores 0-100

**How to use for AEO:**

**Test key pages:**
- Homepage
- Top service pages
- Top blog posts

**Target scores:**
- Mobile: 80+
- Desktop: 80+

**Common fixes:**
- Compress images (use TinyPNG - free)
- Enable browser caching (via hosting or plugin)
- Minify CSS/JS (via plugin)
- Use lazy loading for images

**Why it's essential:**
Slow pages = lower rankings, fewer AI citations. Speed is a ranking factor.

## **Keyword Research Tools (Free)**

### Tool 6: Google Keyword Planner

**Cost:** Free (requires Google Ads account, no spend required)
**URL:** https://ads.google.com/home/tools/keyword-planner

**What it does:**
- Shows search volume for keywords
- Suggests related keywords
- Shows competition level
- Provides cost-per-click data

**How to use for AEO:**

**Discover question keywords:**
1. Enter your core service
2. Add "how", "what", "cost", "vs"
3. Filter: 50-500 searches/month
4. Export to spreadsheet
5. Prioritize by commercial intent

**Example for "crane rental":**
- "how much crane rental cost" (320 searches/month)
- "crane rental vs purchase" (150 searches/month)
- "what is minimum crane rental period" (80 searches/month)

**Why it's valuable:**
Google's own search volume data. Most accurate available.

### Tool 7: Google Trends

**Cost:** Free
**URL:** https://trends.google.com

**What it does:**
- Shows trending topics
- Compares keyword popularity over time
- Regional search interest
- Related queries

**How to use for AEO:**

**Find trending questions:**
1. Enter your industry + "questions"
2. Filter to "Malaysia" (last 12 months)
3. Identify rising queries
4. Create content targeting these

**Example:**
- Search: "construction equipment" + "questions"
- Filter: Malaysia, rising
- Result: "crane rental price 2026" up 300%
- Action: Create pricing guide for 2026

**Why it's valuable:**
Get ahead of trends. Create content before competitors.

### Tool 8: AnswerThePublic (Free Tier)

**Cost:** Free (3 searches/day)
**URL:** https://answerthepublic.com

**What it does:**
- Visual question map
- Groups questions by preposition (who, what, where, when, why, how)
- Shows search volume
- Question keyword goldmine

**How to use for AEO:**

**Daily workflow:**
1. Use your 3 free searches
2. Enter core keywords
3. Export question map
4. Prioritize by search volume
5. Add to content calendar

**Example output:**
```
Keyword: "crane rental"

Who questions:
- Who rents cranes? (40 searches)
- Who needs crane rental license? (30 searches)

What questions:
- What is crane rental process? (150 searches)
- What documents needed for crane rental? (80 searches)

How questions:
- How to book crane rental? (200 searches)
- How much to rent crane? (320 searches)
```

**Turn into content:**
- "How to Book Crane Rental in 3 Steps"
- "Crane Rental Process: Complete Guide"
- "Documents Needed for Crane Rental"

**Why it's valuable:**
Question keywords = AI Overview magnet. This tool finds them all.

### Tool 9: Google "People Also Ask"

**Cost:** Free
**URL:** N/A (part of Google search results)

**What it does:**
- Shows related questions in search results
- Expands when clicked (4-6 questions)
- Real user queries

**How to use for AEO:**

**Manual research process:**
1. Search your target keyword
2. Find "People Also Ask" box
3. Click each question
4. Note all questions that appear
5. Add to FAQ section

**Example:**
Search: "crane rental KL"

People Also Ask:
- How much does it cost to rent a crane?
- What is the minimum rental period?
- Do I need insurance for crane rental?
- How is crane rental price calculated?

**Use these for:**
- FAQ schema (exactly what AI Overviews want)
- Blog post topics
- Section headers in articles

**Why it's valuable:**
Google tells you what questions users ask. Use them.

## **Content Optimization Tools (Free)**

### Tool 10: Hemingway Editor

**Cost:** Free (web version)
**URL:** https://hemingwayapp.com

**What it does:**
- Analyzes readability
- Highlights complex sentences
- Suggests simpler alternatives
- Grade level scoring

**How to use for AEO:**

**Optimize content for AI Overviews:**
- Target: Grade 6-8 reading level
- Keep sentences short (average 15-20 words)
- Remove passive voice
- Eliminate adverbs
- Use simple words

**Example:**
- Before: "The utilization of heavy machinery rental services is predicated upon..."
- After: "Crane rental works best when..."

**Why it's valuable:**
AI engines prefer simple, direct language. Hemingway helps achieve this.

### Tool 11: Google Docs

**Cost:** Free (with Google account)
**URL:** https://docs.google.com

**What it does:**
- Word processor
- Word count
- Collaboration
- Version history

**How to use for AEO:**

**Content creation workflow:**
1. Create document
2. Write content
3. Use "Word count" to check length (target 1,000-1,200)
4. Share with team for review
5. Version history tracks changes
6. Copy-paste to website when done

**Pro tip:**
Use Google Docs "Outline" view to ensure proper H2/H3 structure before publishing.

**Why it's valuable:**
Simple, free, effective. No need for paid content tools.

**Cost:** Free (basic features)

**What it does:**
- Graphic design tool

**How to use for AEO:**

**Create featured images:**
- Template: Blog graphic (1200x630px)
- Add headline (keyword)
- Add brand elements
- Export as JPG/PNG
- Upload to website

**Create social images:**
- Quote graphics from your content
- Share on LinkedIn, Facebook
- Drive traffic back to blog

**Why it's valuable:**
Professional images without designer. Free tier sufficient for most needs.

**What it does:**
- Website crawler
- Finds broken links
- Analyzes meta data
- Checks redirects
- Identifies technical issues

**How to use for AEO:**

**Monthly technical audit:**
1. Enter your website URL
2. Run crawl (free version: up to 500 pages)
3. Check for:
   - [ ] 404 errors (fix or redirect)
   - [ ] Missing title tags
   - [ ] Missing meta descriptions
   - [ ] Duplicate content
   - [ ] Redirect chains

**Fix top 10 issues each month.**

**Why it's valuable:**
Technical issues kill SEO. This tool finds them.

### Tool 14: Google Mobile-Friendly Test

**Cost:** Free
**URL:** https://search.google.com/test/mobile-friendly

**What it does:**

**How to use for AEO:**
