## 6. Content Quality Standards

### Minimum Requirements
- **Word count:** 1500-2500 words
- **Readability:** Flesch 60+ (Grade 8-9)
- **Structure:** H1 + 4-6 H2s + H3s
- **Keywords:** Primary keyword density 1-2%
- **FAQ section:** 5-7 questions
- **Links:** 3-5 internal, 2-3 external
- **Images:** 2-4 relevant images with alt text
- **CTA:** 3-5 conversion points

### SEO Optimization Checklist
- [ ] Title tag optimized (50-60 chars)
- [ ] Meta description (150-160 chars)
- [ ] H1 includes primary keyword
- [ ] Primary keyword in first paragraph
- [ ] LSI keywords naturally integrated
- [ ] Internal links to related content
- [ ] External links to authoritative sources
- [ ] Schema markup (Article + FAQPage)
- [ ] Alt text for all images
- [ ] Mobile-responsive formatting
