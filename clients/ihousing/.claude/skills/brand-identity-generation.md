# Brand Identity (CI) Generation Skill

## Purpose
Create comprehensive Corporate Identity (CI) options for clients who don't have established brand guidelines, based on logo analysis or industry best practices.

## Key Principle
**Generate 5 distinct CI options** with complete visual specifications, easy for client to review and choose.

## Input Sources

### If Client Has Logo:
- Analyze existing logo file
- Extract colors, style, typography hints
- Build CI around existing brand mark
- Ensure consistency with established logo

### If Client Has NO Logo:
- Use industry context (construction, logistics, etc.)
- Use cultural context (Malaysia, target audience)
- Use competitor analysis
- Create modern, professional CI from scratch

- Logo style (if exists)

- Target audience (B2B, B2C, B2G)
- Cultural context (Malaysia, multilingual)

- **Option 1:** Logo-based (if logo exists) OR Professional
- **Option 2:** Modern Minimal

- **Option 5:** Warm Approachable

For each option:
- Homepage hero section
- Service/product card
- Blog post layout
- CTA button examples
- Color distribution diagrams

### Step 4: Provide Selection Guidance
Help client choose by explaining:
- What each option conveys
- Who it appeals to
- Industry alignment
- Strengths and trade-offs
