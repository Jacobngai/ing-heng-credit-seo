## 物业管理费基础知识

### 什么是物业管理费？

**物业管理费**（Maintenance Fee）是公寓业主每月向物业管理公司支付的费用，用于维护和管理公共区域和设施。

**管理费包含：**

| 项目 | 说明 | 典型成本占比 |
|-----|------|-------------|
| **保安服务** | 24小时保安、CCTV | 25-30% |
| **清洁服务** | 公共区域清洁 | 15-20% |
| **垃圾处理** | 垃圾收集处理 | 5-8% |
| **公共设施** | 泳池、健身房维护 | 20-25% |
| **电梯维护** | 检查、维修 | 10-15% |
| **水电公摊** | 公共区域水电 | 8-12% |
| **行政费用** | 管理公司运营 | 10-15% |
| **维修储备金** | 长期维修基金 | 5-10% |

### 不包含在管理费中的费用

**业主需要自行承担的费用：**

| 费用类型 | 说明 | 典型金额/月 |
|---------|------|------------|
| **评估费** | Quit Rent（地税）| RM50-150 |
| **门牌税** | Assessment（市税）| RM30-100 |
| **火险保险** | 房贷要求 | RM100-200 |
| **单位水电** | 客人使用 | RM150-500 |
| **WiFi网络** | 短租必需 | RM100-200 |
| **维修费用** | 单位内部维修 | RM50-200 |

**月度总固定成本：**
- 管理费：RM200-500
- 评估税+门牌税：RM80-250
- 保险：RM100-200
- WiFi：RM100-200
- **总计：RM480-1,150/月**

## 管理费常见问题

### Q1：管理费会涨价吗？

**答：** 可能，但有规定：
- 通常每年调整一次
- 涨幅需要业主委员会批准
- 涨幅通常5-10%
- 查看历史涨价记录

### Q2：管理费包含维修吗？

**答：** 分两种情况：
- **公共区域维修**：包含在管理费中
- **单位内部维修**：业主自费
- 大型维修可能需要特殊评估

### Q3：如果不付管理费会怎样？

**答：** 后果严重：
- 利息累积（通常10%/年）
- 法律行动
- 不能出售房产
- 影响信用记录

### Q4：管理费可以协商吗？

**答：** 不能，但可以：
- 参加业主委员会
- 影响费用使用
- 监督管理质量
- 争取更好服务

### Q5：如何降低管理费影响？

**答：** 三种方法：
1. 提高收入（优化定价）
2. 降低成本（能源节约）
3. 混合模式（短租+长租）

### 管理费影响计算器

**公式：**

```
月度净收入 = 月度收入 - 管理费 - 其他成本

管理费占比 = 管理费 ÷ 月度收入

目标：管理费占比 < 15%
```

**示例：**
```
月度收入：RM3,000
管理费：RM350
其他成本：RM500

净收入：RM2,150
管理费占比：11.7%（健康）

如果管理费RM500：
净收入：RM2,000
管理费占比：16.7%（偏高）
```
