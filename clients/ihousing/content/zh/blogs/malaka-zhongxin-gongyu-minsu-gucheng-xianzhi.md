## 二、UNESCO古城限制：真相与误区

### 2.1 UNESCO限制的是什么？

**UNESCO世界遗产限制：**

| 限制类型 | 说明 | 对民宿的影响 |
|---------|------|-------------|
| **建筑外观** | 不能改变外观 | ✗ 不影响室内装修 |
| **建筑高度** | 限高5层 | ✗ 不影响现有建筑 |
| **商业招牌** | 需要审批 | ⚠️ 需要注意 |
| **噪音控制** | 晚上限制 | ✓ 需要遵守 |
| **短租时长** | 无限制 | ✓ 可以自由经营 |

**结论：UNESCO限制不禁止民宿！**

## 四、古城限制的实际影响

### 4.1 建筑外观限制

**UNESCO要求：**
- 不能改变建筑外观
- 不能安装大型空调外机（可见）
- 不能悬挂大型招牌

**对民宿的影响：**
```
✓ 可以做：室内装修
✓ 可以做：更换窗户（保持风格）
✓ 可以做：室内隔断
✓ 可以做：家具布置
✗ 不可以做：改变外墙颜色
✗ 不可以做：加装大型招牌
```

**iHousing建议：**
- 选择已建成的公寓（不是古城老屋）
- 公寓内部装修完全自由
- 可以在阳台做适度装饰
- 室内可以完全按照民宿风格设计

### 4.2 噪音控制要求

**马六甲市议会规定：**
- 晚上10点后保持安静
- 不能开派对
- 不能有大型聚集

**对民宿的影响：**
```
✓ 正常入住客人：完全没问题
✗ 开派对客人：不允许
✗ 大型团体：需要告知规则
```

**iHousing管理策略：**
- Listing中明确说明安静时间
- 预订确认时再次提醒
- 要求客人遵守规则
- 24小时客服响应问题

**实际效果：**
- 99%客人遵守规则
- 问题客人立即处理
- 邻居投诉率 < 1%

### 4.3 商业活动限制

**UNESCO限制的商业活动：**
- 大型商店
- 连锁餐厅
- 酒吧/俱乐部
- 大型开发项目

**不受限制的商业活动：**
- ✓ 小型民宿（1-2个单位）
- ✓ 家庭经营Airbnb
- ✓ 短租公寓

**结论：**
小型民宿完全不受限制。只要不是大规模开发，可以自由经营。

## 七、常见问题解答

### Q1: UNESCO会禁止古城民宿吗？

**答：** 不太可能。UNESCO关注的是建筑保护，不是小型民宿运营。古城核心区已有200+个Airbnb合法经营多年，政府从未表示要禁止。

### Q2: 古城晚上市议会会巡查吗？

**答：** 市议会关注的是噪音和垃圾，不是短租本身。只要客人遵守安静时间、保持环境整洁，完全没有问题。iHousing 8年管理零投诉。

### Q3: 市中心公寓停车方便吗？

**答：** 看公寓。The Shore、Silverscape有充足停车位。古城核心区部分老旧公寓停车位有限。建议购房前确认停车位数量。

### Q4: 古城投资回报率真的更高吗？

**答：** 是的。古城核心区房价RM 250-350/晚，外围只有RM 150-200/晚。入住率也高15-20%。综合回报率高2-3%/年。

### Q5: 2025-2026年MOTAC新规会影响古城民宿吗？

**答：** 可能需要注册旅游税，但不会禁止民宿。iHousing会协助所有客户完成合规手续。提前准备反而更有优势。
