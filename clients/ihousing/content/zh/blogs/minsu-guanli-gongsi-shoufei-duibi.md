## 盈亏平衡分析

### 固定费用何时更划算？

**盈亏平衡点计算：**

```
固定费用 = 佣金率 × 收入
收入 = 固定费用 ÷ 佣金率
```

**20%佣金模式：**
```
盈亏平衡点 = RM 250/月 ÷ 20%
            = RM 1,250/月
            = RM 15,000/年
```

**结论：** 如果您的年预计收入 > RM 15,000，固定费用更划算！

**实际情况：**
- 马六甲2房公寓平均收入：RM 30,000-50,000/年
- 远高于盈亏平衡点RM 15,000
- **固定费用几乎总是更划算！**
