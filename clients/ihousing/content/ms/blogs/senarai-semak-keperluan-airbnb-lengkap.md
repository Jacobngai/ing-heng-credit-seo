# Senarai Semak Keperluan Airbnb: Apa Perlu Disediakan 2025

## Senarai Semak Persediaan Sebelum Daftar Masuk

### 1 Minggu Sebelum
- [ ] Pesan barang habis guna (sabun, syampu, tisu)
- [ ] Semak semua peralatan berfungsi
- [ ] Uji pintar berkunci
- [ ] Semak WiFi berfungsi
- [ ] Bersihkan unit sepenuhnya

### 1 Hari Sebelum
- [ ] Ganti lentik & handuk
- [ ] Isi semula barang habis guna
- [ ] Letak minuman selamat datang
- [ ] Pastikan suhu bilik selesa
- [ ] Uji semua lampu & AC

### Hari Daftar Masuk
- [ ] Semak kebersihan bilik air
- [ ] Pastikan tiada barang peribadi tertinggal
- [ ] Hantar mesej selamat datang
- [ ] Sediakan kata laluan WiFi
- [ ] Pastikan manual rumah ada

## Tips Jimat Wang

### Tip 1: Guna Barang Terpakai untuk Item Tertentu
Boleh beli terpakai:
- Kerusi, meja
- Perabot dapur
- Hiasan dinding
- Rak

JANGAN beli terpakai:
- Tilam (alasan kesihatan)
- Lentik/handuk
- Peralatan elektrik

### Tip 2: Mulakan dengan Asas
Beli item wajib dulu:
- Katil + tilam: RM2,000
- Sofa: RM800
- Dapur asas: RM1,500
- TV: RM1,000
- Pintar berkunci: RM600

Jumlah permulaan: RM5,900

### Tip 3: Beli Set Berbanding Berasingan
Contoh jimat:
- Set meja makan (meja + 4 kerusi) = RM800 vs RM1,000 (beli berasingan)
- Set bilik tidur (katil + 2 almari) = RM3,000 vs RM3,800

### Tip 4: Prioritaskan "Touch Points"
Item yang banyak disentuh tetamu:
- Lentik & handuk berkualiti
- Sofa selesa
- Peralatan dapur stainless steel
- Tombol pintu metal
