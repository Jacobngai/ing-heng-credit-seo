```css
:root {
  --primary: #...;
  --secondary: #...;
  /* etc. */
}
```

### Step 3: Update Tailwind Config
Apply colors to `clients/ing-heng/website/tailwind.config.js`

### Step 4: Update Components
Replace color classes in all Astro components:
- `bg-primary`, `text-primary`, `border-primary`
- Button styles
- Hero sections
- Cards and layouts

### Step 5: Test Accessibility
Verify WCAG AA contrast ratios:
- Primary text on backgrounds
- Button text on button colors
- Link colors on backgrounds

### Step 6: Rebuild & Deploy
```bash
npm run build
vercel --prod
```
