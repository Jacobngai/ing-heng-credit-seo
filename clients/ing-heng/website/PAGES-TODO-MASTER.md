## How Sitemap Works

When a page is created (e.g., `/en/blog/excavator-financing.astro`):
1. ✅ It's automatically added to sitemap.xml by @astrojs/sitemap plugin
2. ✅ lastmod is set to today's date
3. ✅ changefreq and priority are set based on page type
4. ✅ Google will discover it via sitemap submission

No manual sitemap update needed!
