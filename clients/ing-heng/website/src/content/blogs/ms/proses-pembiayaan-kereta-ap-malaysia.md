### Realiti Permit AP

**Apakah Kereta AP?**
Kereta AP (Approved Permit) adalah kenderaan yang diimport di bawah sistem import rasmi Malaysia. Anda bayar:
- Import duty (sehingga 105%)
- Excise duty (sehingga 125%) 
- Sales tax (10%)
- Yuran pemprosesan kastam

**Jumlah kos boleh 200-300% dari nilai asal kereta.**

Contoh: Kereta RM150,000 jadi RM450,000 selepas duti.

Namun bank tolak pembiayaan kerana mereka tak faham proses import yang sah.

## Corak Penolakan Bank Untuk Kereta AP

### Ketakutan Kerumitan Dokumentasi

**Kebimbangan Bank:**
"Dokumen-dokumen import ni terlalu rumit. Bagaimana kalau ada isu undang-undang?"

**Realiti:**
Permit AP diluluskan kerajaan. Dokumentasi sah termasuk:
- Permit AP dikeluarkan MITI
- Pelepasan Royal Customs
- Kelulusan pendaftaran JPJ
- Sijil pemeriksaan Puspakom
- Resit bayaran import duty

Bank takut dengan apa yang mereka tak faham.

### Andaian Umur dan Keadaan

**Logik Bank:** "Kereta import lama dan bermasalah."

**Realiti:** Banyak import AP adalah:
- 2-5 tahun (hampir baru)
- Diselenggara lebih baik dari setara pasaran tempatan
- Model spesifikasi tinggi tak tersedia tempatan
- Sering mileage rendah kerana keadaan jalan yang lebih baik di luar negara

### Kerisauan Ubah Suai dan Pematuhan

**Ketakutan Bank:** "Kereta import mungkin tak patuh dengan standard Malaysia."

**Realiti:**
- Semua kereta AP lulus pemeriksaan Puspakom
- Patuh dengan standard keselamatan Malaysia
- Diubah suai untuk keperluan tempatan (laraskan lampu depan, dll.)
- Sah untuk jalan raya dan boleh diinsurans

### Kebimbangan Insurans dan Rangkaian Servis

**Andaian Bank:** "Kereta import susah untuk insurans dan servis."

**Realiti:**
- Beberapa insurer cover import AP
- Pusat servis sah uruskan varian import
- Spare part tersedia melalui dealer rasmi
- Nilai sering lebih tinggi dari setara pasaran tempatan
