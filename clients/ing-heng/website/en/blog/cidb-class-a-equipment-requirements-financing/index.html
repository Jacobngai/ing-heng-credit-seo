Understanding CIDB Classification & Equipment Requirements

The Construction Industry Development Board (CIDB) classification system determines which contracts you can bid on. The difference between Class B and Class A isn‚Äôt just prestige - it‚Äôs millions in tender opportunities.

Class A Benefits:

CIDB Equipment Point System Explained

How Equipment Points Work:

Each equipment type carries specific CIDB points based on:

  • Equipment capacity (horsepower, tonnage, lifting capacity)
  • Specialization level (basic vs advanced functionality)
  • Age and condition (newer equipment may carry more points)
  • Certification status (DOSH compliance, etc.)

Major Equipment Categories & Points:

Heavy Earthmoving (High Points):

  • Bulldozers D7-D9: 25-40 points each
  • Large excavators (30+ ton): 20-35 points each
  • Wheel loaders (966-980 class): 15-25 points each
  • Motor graders (14M-16M): 20-30 points each

Lifting & Handling (Premium Points):

  • Mobile cranes (50+ ton): 30-50 points each
  • Tower cranes: 35-60 points each
  • Concrete pumps (truck-mounted): 25-40 points each
  • Crawler cranes (100+ ton): 40-70 points each

Specialized Production (Maximum Points):

  • Crushing plants: 80-150 points each
  • Asphalt plants: 100-200 points each
  • Concrete batching plants: 60-120 points each
  • Piling rigs: 40-80 points each

Standard Construction (Base Points):

  • Backhoe loaders: 8-15 points each
  • Compactors: 5-12 points each
  • Tipper trucks: 3-8 points each
  • Prime movers: 10-18 points each
