## 14. Technical SEO Specifications

### Image Optimization Standards
- **Format**: WebP (primary), JPG (fallback)
- **Compression**: 80-85% quality (balance size/quality)
- **Dimensions**: Multiple sizes (hero, card, thumbnail, mobile)
- **Alt Text**: Keyword-rich, descriptive, multilingual
- **Lazy Loading**: Applied to below-fold images
- **File Naming**: SEO-friendly (e.g., `komatsu-pc350-excavator-financing-malaysia.jpg`)

### On-Page SEO
- **Meta Titles**: 50-60 characters, keyword-optimized
- **Meta Descriptions**: 150-160 characters, CTA included
- **H1 Tags**: Single, keyword-focused
- **H2-H6**: Structured hierarchy
- **Schema Markup**: Product, Offer, Organization schemas

### Technical Requirements
- **Page Speed**: Target 90+ on PageSpeed Insights
- **Mobile-First**: Responsive design, touch-friendly
- **Core Web Vitals**: LCP <2.5s, FID <100ms, CLS <0.1
- **SSL**: HTTPS only
- **Sitemap**: XML sitemap with image sitemap
