Usage Guidelines

Do's ‚úÖ

  • Use bold, asymmetric layouts for visual interest
  • Incorporate geometric shapes and patterns
  • Use gradient overlays on images (subtle, 10-20%)
  • Animate interactive elements (hover, click)
  • Use duotone icons for modern feel
  • Leverage white space for breathing room

Don'ts ‚ùå

  • Don't use more than 3 colors per section
  • Don't create overly complex layouts
  • Don't use heavy shadows (keep them soft)
  • Don't use outdated patterns (old gradients)
  • Don't sacrifice readability for style
  • Don't forget mobile-first approach
